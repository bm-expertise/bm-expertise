ПРОМЫШЛЕННАЯ БЕЗОПАСНОСТЬ

Экспертиза промышленной безопасности и разработка технической документации

ЭКСПЕРТИЗА ПРОМЫШЛЕННОЙ БЕЗОПАСНОСТИ

один из видов экспертизы документации, которая проводится для опасных производственных объектов, с целью определения соответствия принятых проектных решений установленным требованиям промышленной безопасности.

ЦЕЛЬ

Целью экспертизы является определение соответствия объекта требованиям промышленной безопасности, проверка соблюдения правил, которые утверждены законодательством РК.

Экспертиза промышленной безопасности необходима в следующих случаях

  • Для экспертизы проектной и строительной документации.
  • Для продления срока службы эксплуатации оборудования и технических устройств.
  • Для оформления «Разрешение на применение» технологий, технических устройств, материалов на опасных производственных объектах.
  • Для разработки деклараций и тех. паспортов устройств.
  • Для определения технического состояния зданий, сооружений.
  • Для получения аттестата юридическому лицу на соответствие заявленным видам работ.

Результат

Результатом экспертизы промышленной безопасности является Экспертное заключение – оценка соответствия (несоответствия) объекта экспертизы предъявляемым к нему требованиям промышленной безопасности и действующей нормативно-технической документации.

РАЗРАБОТКА ТЕХНИЧЕСКОЙ ДОКУМЕНТАЦИИ

совокупность материалов и документов, обеспечивающих качество производимой продукции, оказываемых услуг, проводимых работ, а также их соответствие всем утвержденным стандартам, требованиям безопасности, условиям эксплуатации, хранения и транспортировки.

Применение

Техническая документация применяется в проектировании, строительстве, энергетике, в области охраны труда и промышленной безопасности. Разработка технической документации – процесс создания набора данных документов.

Виды технической документации

  • Паспорт (формуляр) технического устройства;
  • Расчеты на прочность;
  • Паспорт безопасности химической продукции
  • Руководство по эксплуатации;
  • Технические условия;
  • Другие документы в зависимости от спецификации продукции, услуг и работ.

Результат

Получение технической документации согласно требованиям действующего законодательства РК.
